# Data Directory

This folder contains reference solutions used for evaluating
GNN-PINN and MLP-PINN predictions.

---

## Files

| File | Shape | Description |
|---|---|---|
| `u_ref_smooth.npy` | [100, 200] | Bimetallic rod displacement field. 100 time steps × 200 spatial nodes. Generated by scipy RK45. |
| `x_nodes_smooth.npy` | [200] | Spatial node positions. Uniform on [0, 1]. |
| `t_eval_smooth.npy` | [100] | Time evaluation points. Uniform on [0, 1]. |

---

## How to Regenerate

These files are generated by running:

```bash
python experiments/1d_bimetallic/step1_reference_smooth.py
```

Runtime: ~20 seconds.

The script solves the wave equation using the method of lines:
- Discretise space into N=200 nodes
- Convert PDE to a system of ODEs in time
- Integrate with scipy RK45 (rtol=atol=1e-8)

---

## Wave Speed Profile

The bimetallic rod uses a smooth sigmoidal wave speed:

```
c(x) = c1 + (c2 - c1) * sigmoid((x - 0.5) / 0.03)

c1 = 1.0  (normalised steel,     x < 0.5)
c2 = 1.5  (normalised aluminium, x > 0.5)
```

Theoretical reflection coefficient: R = (c2-c1)/(c2+c1) = 0.20
Theoretical transmission coefficient: T = 2c1/(c2+c1) = 0.80

## Loading the Data

```python
import numpy as np
from scipy.interpolate import RegularGridInterpolator

u_ref  = np.load('data/u_ref_smooth.npy')    # [100, 200]
x_ref  = np.load('data/x_nodes_smooth.npy')  # [200]
t_ref  = np.load('data/t_eval_smooth.npy')   # [100]

# Interpolate to any (x, t) point
interp = RegularGridInterpolator(
    (t_ref, x_ref), u_ref,
    method='linear', bounds_error=False)

# Example: evaluate at (x=0.5, t=0.3)
u_val = interp([[0.3, 0.5]])
print(f"u(0.5, 0.3) = {u_val[0]:.5f}")
```
